# Mystery of Antiques

PDF and EPUB archive for *Mystery of Antiques*, updated alongside AO3 chapter releases.

## Download

- **[Mystery of Antiques.pdf](https://github.com/Anecno/antique/releases/latest/download/Mystery.of.Antiques.pdf)** — recommended. Fixed layout, faithful to the original typesetting.
- [Mystery of Antiques.epub](https://github.com/Anecno/antique/releases/latest/download/Mystery.of.Antiques.epub) — reflowable, with embedded fonts. Best read in [Calibre](https://calibre-ebook.com/download) on desktop, or [KOReader](https://f-droid.org/packages/org.koreader.launcher.fdroid/) on Android phones and e-readers (use the **Download APK** button on that page). If the text looks too large or too small, just adjust the font size in your reader — the whole book scales with it.
